# 3-Phase Input Choke / Reactor Design (SolidWorks)

## 📌 Overview

This project presents the **3D design and modeling of a 3-phase input choke (reactor)** used in power electronics applications.
The model is developed in **SolidWorks** based on real engineering design parameters including winding, core, insulation, and assembly.

---

## ⚙️ Key Specifications

* Frequency: 50 Hz
* Inductance: 1.71 mH
* Current: 43 A
* Core Type: Laminated CRNO core
* Conductor: Copper strip (6 × 3 mm)
* Number of Coils: 3
* Turns per Coil: 30
* Winding Type: Strip winding (multi-layer)
* Cooling: Air duct (8 mm internal duct)

---

## 🧩 Features of the Model

* Accurate **core geometry (limb & yoke)**
* Detailed **strip winding design**
* Multi-layer winding with **inter-layer insulation**
* **Internal cooling duct** modeling
* Proper **lead routing (start & finish terminals)**
* Complete **3-phase assembly**
* Mechanical structure with **clamping plates & bolts**

---

## 🛠️ Tools Used

* SolidWorks (3D Modeling & Assembly)
* Engineering calculations for winding and spacing

---

## 📐 Design Highlights

* Implemented **real-world winding spacing (6.13 mm)**
* Modeled using **sweep + body pattern technique**
* Included **electrical and thermal considerations**
* Designed based on **practical manufacturable approach**
